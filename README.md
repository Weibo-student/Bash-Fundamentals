# Bash-Fundamentals

## What is Bash
Bash (Bourne Again Shell) is a **command-line interpreter** and **scripting language**.

It's the **default** user interface for Unix-based operating systems, like MacOS or Linux.

The terminal we have on the Debian is using the bash as its command line interface.
